1. 服務說明

1.1 服務內容

番茄鱷 是一款學習時間管理應用程式，提供以下功能：

  • 學習計時器（自由模式和番茄鐘模式）
  • 學習記錄和統計分析
  • 學習圈社群功能
  • 學習排行榜
  • 學習目標和成就系統

1.2 服務類型

本應用程式提供免費版和付費訂閱版（番茄鱷 Pro）。

2. 帳戶註冊和安全

2.1 帳戶建立

2.2 帳戶責任

  • 您對使用您帳戶進行的所有活動負責

  • 我們不對因未經授權訪問您帳戶造成的任何損失負責

2.3 帳戶終止

  • 您可以隨時刪除您的帳戶

3. Pro 會員訂閱

3.1 訂閱服務

番茄鱷 Pro 提供增強功能，包括但不限於：

  • 完整歷史資料存取
  • 無限學習圈加入
  • 專注自律模式
  • 學習目標設定
  • 成就系統
  • Pro 專屬頭像框和徽章
  • 進階統計分析

3.2 訂閱價格和期限

訂閱類型計費週期價格
月度會員每月以 App Store 顯示為準
年度會員每年以 App Store 顯示為準

實際價格以您所在地區 App Store 顯示的當地貨幣價格為準。

3.3 訂閱條款

付款

  • 訂閱費用在購買確認時會從您的 Apple ID 帳戶扣款
  • 所有費用以當地貨幣計價，可能因地區而異

自動續訂

  • 除非您在當前訂閱期結束前至少 24 小時取消，否則訂閱將自動續訂
  • 續訂費用將在當前訂閱期結束前 24 小時內從您的帳戶扣款
  • 續訂費用等於當前訂閱價格

管理訂閱

購買後，您可以隨時透過 Apple ID 帳戶設定管理或取消訂閱：

  1. 打開 iPhone「設定」
  2. 點擊您的姓名
  3. 點擊「訂閱項目」
  4. 選擇 番茄鱷 Pro
  5. 選擇取消訂閱或更改訂閱方案

免費試用

  • 如果提供免費試用，試用期結束後將自動轉為付費訂閱
  • 在試用期內取消可避免扣款
  • 購買訂閱後，未使用的免費試用期將被沒收

3.4 退款政策

  • 所有訂閱費用由 Apple 處理
  • 如需退款請求，請直接聯繫 Apple 支援
  • 在活躍訂閱期內取消不提供按比例退款

3.5 恢復購買

如果您更換裝置或重新安裝應用程式，可以透過「恢復購買」功能恢復您的 Pro 會員狀態。
