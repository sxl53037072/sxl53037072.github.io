FocusCroc 이용약관

최종 업데이트: 2025년 12월 17일

시행일: 2025년 12월 17일

다음 이용약관(이하 "본 약관")을 주의 깊게 읽어주십시오. FocusCroc 애플리케이션(이하 "본 앱")을 사용함으로써 본 약관에 구속되는 것에 동의하게 됩니다. 본 약관의 어느 부분에도 동의하지 않는 경우 본 앱을 사용하지 마십시오.

1. 서비스 설명

1.1 서비스 내용

FocusCroc은 다음 기능을 제공하는 학습 시간 관리 애플리케이션입니다:

  • 학습 타이머(자유 모드 및 뽀모도로 모드)
  • 학습 기록 및 통계 분석
  • 학습 서클 커뮤니티 기능
  • 학습 순위표
  • 학습 목표 및 성취 시스템

1.2 서비스 유형

본 앱은 무료 버전과 유료 구독 버전(FocusCroc Pro)을 제공합니다.

2. 계정 등록 및 보안

2.1 계정 생성

  • 본 앱을 사용하려면 13세 이상이어야 합니다
  • 진실하고 정확하며 완전한 등록 정보를 제공해야 합니다
  • 계정 자격 증명의 기밀성을 유지할 책임이 있습니다

2.2 계정 책임

  • 귀하의 계정을 사용하여 수행되는 모든 활동에 대해 책임을 집니다
  • 계정의 무단 사용을 발견하면 즉시 알려주십시오
  • 계정에 대한 무단 접근으로 인한 손실에 대해서는 책임을 지지 않습니다

2.3 계정 종료

  • 언제든지 계정을 삭제할 수 있습니다
  • 본 약관 위반에 대해 계정을 정지하거나 종료할 권리를 보유합니다

3. Pro 멤버십 구독

3.1 구독 서비스

FocusCroc Pro는 다음을 포함한 향상된 기능을 제공합니다:

  • 전체 기록 데이터 접근
  • 무제한 학습 서클 참여
  • 집중 자기 규율 모드
  • 학습 목표 설정
  • 성취 시스템
  • Pro 전용 아바타 프레임 및 배지
  • 고급 통계 분석

3.2 구독 가격 및 기간

구독 유형청구 기간가격
월간 멤버십매월App Store 표시 가격
연간 멤버십매년App Store 표시 가격

실제 가격은 귀하 지역의 App Store에서 현지 통화로 표시됩니다.

3.3 구독 조건

결제

  • 구매 확인 시 Apple ID 계정에 구독료가 청구됩니다
  • 모든 요금은 현지 통화로 책정되며 지역에 따라 다를 수 있습니다

자동 갱신

  • 현재 구독 기간 종료 최소 24시간 전에 취소하지 않으면 구독이 자동으로 갱신됩니다
  • 현재 기간 종료 24시간 이내에 계정에 갱신 요금이 청구됩니다
  • 갱신 요금은 현재 구독 가격과 동일합니다

구독 관리

구매 후 Apple ID 계정 설정에서 언제든지 구독을 관리하거나 취소할 수 있습니다:

  1. iPhone "설정" 열기
  2. 이름 탭
  3. "구독" 탭
  4. FocusCroc Pro 선택
  5. 구독 취소 또는 플랜 변경 선택

무료 체험

  • 무료 체험이 제공되는 경우 체험 기간 종료 후 자동으로 유료 구독으로 전환됩니다
  • 체험 기간 중 취소하면 요금이 청구되지 않습니다
  • 구독 구매 시 사용하지 않은 무료 체험 기간은 무효화됩니다

3.4 환불 정책

  • 모든 구독료는 Apple에서 처리합니다
  • 환불 요청은 Apple 지원에 직접 문의하십시오
  • 활성 구독 기간 중 취소에 대한 일할 환불은 없습니다

3.5 구매 복원

기기를 변경하거나 앱을 다시 설치하는 경우 "구매 복원" 기능을 통해 Pro 멤버십 상태를 복원할 수 있습니다.

4. 사용자 행동

4.1 금지된 행위

본 앱을 사용할 때 다음 행위를 하지 않을 것에 동의합니다:

  • 서비스 남용: 불법 활동에 앱 사용
  • 서비스 방해: 앱의 정상적인 작동을 방해하거나 중단하려는 시도
  • 허위 정보: 허위 또는 오해의 소지가 있는 정보 게시
  • 다른 사용자 괴롭힘: 학습 서클에서 다른 사용자를 괴롭히거나 모욕하거나 위협
  • 부정 행위: 자동화 도구나 기타 방법을 사용하여 학습 데이터 위조
  • 권리 침해: 다른 사람의 지적 재산권이나 기타 법적 권리 침해
  • 무단 접근: 앱 시스템이나 다른 사용자 계정에 무단으로 접근하려는 시도

4.2 사용자 콘텐츠

  • 학습 서클에 게시하는 콘텐츠에 대해 책임을 집니다
  • 본 약관을 위반하는 콘텐츠를 삭제할 권리를 보유합니다
  • 서비스 제공에 필요한 범위 내에서 게시된 콘텐츠를 사용할 수 있는 라이선스를 당사에 부여합니다

5. 지적 재산권

5.1 앱 소유권

  • 본 앱과 모든 콘텐츠, 기능 및 디자인은 RikSong의 재산입니다
  • 저작권법, 상표법 및 기타 지적 재산권법의 보호를 받습니다
  • 서면 허가 없이 복제, 수정, 배포 또는 기타 방법으로 사용할 수 없습니다

5.2 사용자 데이터

  • 학습 데이터의 소유권은 귀하에게 있습니다
  • 필요에 따라 데이터를 저장, 처리 및 표시할 수 있는 제한적 라이선스를 당사에 부여합니다
  • 이 라이선스는 계정 삭제 시 종료됩니다

6. 면책 조항

6.1 "있는 그대로" 서비스 제공

본 앱은 "있는 그대로" 및 "이용 가능한 상태"로 제공되며, 다음을 포함한 명시적이거나 묵시적인 보증이 없습니다:

  • 상품성 또는 특정 목적에의 적합성
  • 중단 없거나 오류 없는 작동
  • 결함 수정

6.2 학습 결과

  • 본 앱은 학습 지원 도구이며 특정 학습 결과를 보장하지 않습니다
  • 학습 결과는 사용자 자신의 노력과 방법에 달려 있습니다
  • 본 앱 사용으로 인한 학습 결과에 대해 책임을 지지 않습니다

6.3 데이터 정확성

  • 데이터 정확성을 보장하기 위해 노력하지만 절대적인 정확성을 보장하지 않습니다
  • 통계 데이터는 참고용으로만 제공됩니다

7. 책임 제한

7.1 손해 배상 제한

법률이 허용하는 최대 범위 내에서 다음에 대해 책임을 지지 않습니다:

  • 간접적, 부수적, 특별, 결과적 또는 징벌적 손해
  • 데이터 손실 또는 손상
  • 이익 손실
  • 사업 중단

7.2 최대 책임

어떠한 경우에도 당사의 총 책임은 지난 12개월 동안 귀하가 지불한 구독료 총액을 초과하지 않습니다.

8.2 통지
