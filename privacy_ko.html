FocusCroc 개인정보 처리방침

최종 업데이트: 2025년 12월 17일

시행일: 2025년 12월 17일

1. 소개

FocusCroc(이하 "본 앱")을 이용해 주셔서 감사합니다. 본 개인정보 처리방침은 당사가 귀하의 개인정보를 어떻게 수집, 사용, 저장 및 보호하는지 설명합니다. 본 앱을 사용함으로써 본 개인정보 처리방침에 설명된 데이터 처리 방식에 동의하는 것으로 간주됩니다.

본 앱은 RikSong이 개발 및 운영합니다. 본 개인정보 처리방침에 대해 질문이 있으시면 본 정책 끝에 제공된 연락처로 문의해 주십시오.

2. 수집하는 정보

2.1 귀하가 제공하는 정보

  • 계정 정보: 계정 등록 시 이메일 주소, 사용자 이름 및 비밀번호를 수집합니다.
  • 프로필 정보: 아바타, 닉네임 및 자기소개를 선택적으로 제공할 수 있습니다.
  • 학습 목표 데이터: 귀하의 학습 목표, 계획 및 관련 정보.

2.2 자동으로 수집되는 정보

  • 학습 기록: 학습 시간, 시작/종료 시간, 사용한 타이머 모드(자유 모드/뽀모도로 모드) 포함.
  • 통계 데이터: 학습 시간 통계, 연속 체크인 일수, 학습 서클 순위 등.
  • 기기 정보: 기기 유형, 운영 체제 버전, 시간대 설정, 언어 설정.
  • 앱 사용 데이터: 귀하가 사용하는 기능과 빈도.

2.3 구독 및 결제 정보

  • 구독 상태: Pro 멤버십 구독 상태, 구독 유형(월간/연간) 및 만료일.
  • 참고: 실제 결제 처리는 Apple에서 처리합니다. 신용카드 번호나 기타 결제 카드 정보는 저장하지 않습니다.

3. 정보 사용 방법

수집된 정보는 다음 목적으로 사용됩니다:

3.1 핵심 서비스 제공

  • 학습 데이터 기록 및 동기화
  • 학습 통계 보고서 생성
  • 학습 서클 기능 및 순위 서비스 제공
  • Pro 멤버십 구독 관리

3.2 서비스 개선

  • 사용자 경험 향상을 위한 앱 사용 패턴 분석
  • 기술적 문제 식별 및 수정
  • 새로운 기능 개발

3.3 커뮤니케이션

  • 중요한 서비스 알림 전송
  • 문의 및 지원 요청에 대한 응답

4. 정보 저장 및 보안

4.1 데이터 저장

  • 로컬 저장: 일부 학습 데이터는 오프라인 사용을 지원하기 위해 기기에 로컬로 저장됩니다.
  • 클라우드 저장: 데이터 동기화 및 학습 서클 기능을 위해 Supabase 클라우드 서비스를 사용하여 데이터를 저장합니다. 서버는 AWS ap-southeast-1(싱가포르)에 위치합니다.

4.2 데이터 보안 조치

귀하의 정보를 보호하기 위해 업계 표준 보안 조치를 구현합니다:

  • 데이터 전송 시 SSL/TLS 암호화 사용
  • 비밀번호 암호화 저장
  • 접근 제어 및 인증 메커니즘 구현
  • 정기적인 보안 감사

4.3 데이터 보존

  • 본 앱 사용 기간 동안 귀하의 데이터를 보존합니다.
  • 계정을 삭제하면 30일 이내에 개인 데이터를 삭제합니다.
  • 일부 익명화된 통계 데이터는 서비스 개선을 위해 보존될 수 있습니다.

5. 정보 공유

귀하의 개인정보를 판매하지 않습니다. 다음 상황에서 정보를 공유할 수 있습니다:

5.1 학습 서클 기능

  • 학습 서클에 가입하면 다른 회원이 귀하의 사용자 이름, 아바타 및 학습 시간 통계를 볼 수 있습니다.
  • 순위표에 귀하의 순위와 학습 데이터가 표시됩니다.

5.2 제3자 서비스 제공업체

  • Supabase: 데이터 저장 및 사용자 인증
  • Apple: Pro 멤버십 구독 및 결제 처리

5.3 법적 요구사항

법률 및 규정에 의해 요구되거나 법적 절차에 대응하는 경우

6. 귀하의 권리

적용되는 데이터 보호법에 따라 귀하는 다음 권리를 가집니다:

6.1 접근권

앱 내에서 프로필과 학습 데이터를 확인할 수 있습니다.

6.2 정정권

프로필 페이지에서 정보를 수정할 수 있습니다.

6.3 삭제권

다음 방법으로 계정과 데이터를 삭제할 수 있습니다:

  • 앱에서 "설정" > "계정" > "계정 삭제"로 이동
  • 이메일로 삭제 요청

6.4 데이터 이동권

학습 데이터 내보내기를 요청할 수 있습니다. 이메일로 문의해 주십시오.

6.5 동의 철회권

언제든지 앱 사용을 중단하거나 계정을 삭제할 수 있습니다.

7. 아동 개인정보

본 앱은 13세 미만 아동을 대상으로 하지 않습니다. 13세 미만 아동의 개인정보를 실수로 수집한 것을 발견하면 즉시 해당 정보를 삭제합니다.

부모 또는 보호자로서 자녀가 당사에 개인정보를 제공했음을 알게 된 경우 연락해 주십시오.

8. 자동 갱신 구독

8.1 구독 조건

  • Pro 멤버십은 Apple App Store 구독을 통해 관리됩니다
  • 월간 및 연간 두 가지 구독 플랜이 있습니다
  • 구매 확인 시 Apple ID 계정에 구독료가 청구됩니다

8.2 자동 갱신

  • 현재 구독 기간 종료 최소 24시간 전에 자동 갱신을 끄지 않으면 구독이 자동으로 갱신됩니다
  • 갱신 요금은 현재 기간 종료 24시간 이내에 청구됩니다
  • 구매 후 App Store 계정 설정에서 구독을 관리하고 취소할 수 있습니다

8.3 구독 취소

구독을 취소하려면:

  1. iPhone "설정" 열기
  2. 이름 탭 > "구독" 탭
  3. FocusCroc Pro 선택
  4. "구독 취소" 탭

9. 제3자 링크

본 앱에는 제3자 웹사이트 또는 서비스에 대한 링크가 포함될 수 있습니다. 이러한 제3자의 개인정보 보호 관행에 대해서는 책임을 지지 않으며, 해당 개인정보 처리방침을 검토하시기 바랍니다.

중요한 변경 사항에 대해서는 앱 내 알림이나 기타 수단을 통해 알려드립니다. 본 앱의 계속 사용은 업데이트된 개인정보 처리방침에 동의하는 것을 의미합니다.
